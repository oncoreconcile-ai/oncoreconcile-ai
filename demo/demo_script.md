# Demo Script

## Scenario

We receive messy oncology records from different sources.

Example:

```text
Cancer Type: NSCLC
Gene: HER2
Variant: Amplification
```

## Demo Flow

1. Open OncoReconcile AI.
2. Enter cancer type, gene, and variant.
3. Click Reconcile.
4. Show canonical output:
   - Non-Small Cell Lung Cancer
   - ERBB2
   - ERBB2 Amplification
5. Show evidence.
6. Show explanation.
7. Show confidence.
8. Show review status.

## Key Message

OncoReconcile AI does not replace clinical judgment. It creates a trusted, explainable, human-governed data foundation for oncology analytics and future AI workflows.
